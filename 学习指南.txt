查看work文件夹27   https://github.com/FengluJin/mygit.git

1.查看远程仓库
	git remote –v

2.添加源仓库地址为远程仓库
	git remote add  <name>  <remote>

3.远程仓库重命名
	git remote rename old new
	例：git remote rename remote dev


4.提交到暂存区
	git  add <pathspec> 
	例：git add index.html

5.提交到本地仓库
	git commit -m "...”	// ...是版本信息或描述等

提交内容到远程仓库
git  push <remote> <branch>

克隆仓库
git  clone <repository>

查看当前文件的状态
git status
